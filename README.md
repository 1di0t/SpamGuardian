# SpamGuardian

**스팸 메시지를 분류하기 위한 머신러닝 및 딥러닝 기반의 웹 애플리케이션입니다.​**

**목적**

이 프로젝트의 주요 목표는 다양한 알고리즘을 활용하여 스팸 메시지를 효과적으로 식별하고 분류하는 것입니다. 

**데이터 수집**

한국우편사업진흥원, 구글 메일에서 데이터를 수집 후 가공했습니다.


**주요 기능**

나이브 베이즈, KNN, LSTM 알고리즘을 활용한 스팸 분류​

Flask 기반의 백엔드 서버​

React를 사용한 사용자 인터페이스​
